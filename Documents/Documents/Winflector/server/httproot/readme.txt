Sample files for Winflector html client customization:
======================================================

httproot_old.zip
----------------
Html example files compatible with older versions of Winflector HTTP server (earlier than 3.9.3.1)

httproot_simple.zip
-------------------
Easy example based on new Winflector HTML client API (WFAPI)

httproot_advanced.zip
---------------------
Full example based on new Winflector HTML client API (WFAPI) for advanced JavaScript programmers


Choose one of above packages and unzip all files to "\Winflector\server\httproot' folder.

Tutorial and WFAPI docs are available at:
http://www.winflector.com/english/support/javascript-winflector-api.html
